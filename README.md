Project Description:
This project is a Java Swing desktop application that simulates the 5 steps of the ISO/IEC 15939 measurement process.

Steps:
1. Profile
2. Define
3. Plan
4. Collect
5. Analyse

Compilation Instructions:
Go to src folder and run:

javac *.java model/*.java

Run Instructions:
java Main
